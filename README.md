# AI Data Analyst Assistant

## Overview

AI Data Analyst Assistant is a Python-based application that automates data analysis using Artificial Intelligence. Users can upload CSV or Excel datasets and instantly generate insights, visualizations, and reports.

## Features

✔ Upload CSV & Excel files
✔ Automatic Data Cleaning
✔ Exploratory Data Analysis (EDA)
✔ Interactive Charts
✔ Statistical Summary
✔ Business Insights using AI
✔ PDF Report Generation

## Technologies Used

- Python
- SQL
- Pandas
- NumPy
- Streamlit
- OpenAI API / Gemini API
- Matplotlib
- Plotly
- Excel
